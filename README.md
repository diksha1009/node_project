# Node_project
###### This is a simple Register and Login Webpage. 
 **Front end**
  1. HTML/CSS
  2. JS
  
 **Back end**
  1. Node.js and Express
  2. Database MySql
  
  #### How to Start
  ```
  npm i
  npm start
  ```
  #### Prerequisites
  *NodeJs and MySql Workbench*
